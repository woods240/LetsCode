批量替换一组元素的属性，轮班上岗

  • 原理介绍
    • 包装 将多个 { 元素 : 属性 }，组合到一个包里，成为集合
    • 系列 包中所有元素的属性值集合，多种情况可定义多个系列
    • 默认系列 从页面中遍历包中的元素，分别读取当前的属性值，作为一个系列
    • 切换系列 将包中所有元素的属性值，同时更新至一个系列
  • 如何使用
    • packageSelector 使用json定义 { id : attr } 集合（id用于找到元素，attr用于更新属性值）
    • seriesCollection 定义多个系列
      • seriesName 系列的名称
      • seriesValues 以packageSelector的方式，定义该系列的值集合（将attr替换成具体值）
    • showSeriesList 提供时机和参数，允许用户构建一个可视的系列集合
      • $element 调用本插件的jQuery对象
      • seriesList 所有系列名称，组成的数组
      • PackageSwitch 实现插件功能的javascript对象
    • addDefault 是否添加默认系列
    • defaultSeriesName 默认系列的名称

注：
1.插件参数的格式必须正确，目前只支持id作为元素的唯一选择器
2.可以通过showSeriesList构建系列切换列表
3.主页用packageSwitch实现了样式主题切换，可参考：框架页
4.与设计模式中的抽象工厂非常相似
