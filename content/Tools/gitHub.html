分布式版本控制：当前流行的开源版本控制工具

  • 常用命令
    • git clone <项目地址> 克隆GitHub上的项目
    • git status 检查文件状态
    • git add <文件路径> 将文件内容 纳入版本库
    • git rm --cached <文件路径>
    • git commit 提交暂存区的文件
    • git commit -a 跳过暂存区，提交所有文件
  • .gitignore（用正则表达式忽略匹配的 文件[夹]）
    • * 匹配任意多个字符
    • [] 匹配[]中枚举的字符
    • ? 匹配一个字符
    • */ 匹配目录
    • ! 取反
