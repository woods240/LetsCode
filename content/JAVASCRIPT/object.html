对象：无序的属性集合

  • 创建
    • 静态 var o = { "name1": "value1", "name2": "value2", ... }
    • 动态 var o = new 构造函数()
  • 调用
    • 静态 o.name1
    • 动态 o["name1"]
  • Object属性和方法
    • o.constructor 引用构造函数
    • o.prototype 引用原型对象
    • o.toString() 转换为字符串
    • o.valueOf() 转换为数字
    • o.hasOwnProperty('属性名') 检查对象自己的属性（Object的除外）
    • A.isPrototypeOf(B) A是否为B的原型对象
  • 自定义类
    • 定义属性 var MyClass = function(x, ...){ this.属性1 = x; ... }
    • 定义方法 MyClass.prototype.方法1 = function(){ ... }
    • MyClass 被定义成类的构造函数
    • this 指代类的实例，在构造函数和方法中被引用
