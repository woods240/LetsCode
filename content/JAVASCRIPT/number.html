数字：处理数字的对象

  • 创建： var n = 0
  • Number静态常量
    • Number.MAX_VALUE 最大的数
    • Number.MIN_VALUE 最小的数
    • Number.NaN 非数字
    • Number.NEGATIVE_INFINITY 负无穷大
    • Number.POSITIVE_INFINITY 正无穷大
  • Number方法
    • n.toFixed(小数位数) 四舍五入表示法
    • n.toExponential(小数位数) 指数计数表示法
    • n.toPrecision(有效数字个数) 有效数字表示法
