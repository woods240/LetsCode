数据类型：计算机通过操作值来运行，它是能够表示并操作的值的类型

  • 三种基本数据类型
    • 数字 量化工具
    • 字符串 用 '_' 或 "_" 包含的文本
    • 布尔值 表示真假
  • 两种小数据类型
    • null 无值
    • undefined 未定义
  • object类型
    • json对象 已命名的值的无序集合
    • 数组 有编号的值的有序集合
    • 函数 具有可执行代码，可通过调用来执行操作

类型转换

  • 数字转字符串
    • 自动转换 var 字符串 = 数字 + '' ;
    • 指定小数位数 var 字符串 = 数字.toFixed(小数位数) ;
    • 指定有效数字位数 var 字符串 = 数字.toPrecision(有效数字位数) ;
    • 指定指数表示中的小数位数 var 字符串 = 数字.toExponential(指数表示中的小数位数) ;
  • 字符串转数字
    • 自动转换 var 数字 = 字符串 - 0 ;
    • 整数转换 var 数字 = parseInt(字符串) ;
    • 小数转换 var 数字 = parseFloat(字符串) ;
  • 布尔值自动转换
    • 布尔值在数字环境 true -> 1, false -> 0
    • 布尔值在字符串环境 true -> "true", false -> "false"
    • 数字在布尔值环境 0或NaN -> false, 其余 -> true
    • 字符串在布尔值环境 空字符串 -> false, 其余 -> true
    • object在布尔值环境 null和undefined -> false, 其余 -> true
