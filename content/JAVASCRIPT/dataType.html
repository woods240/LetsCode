数据类型：计算机通过操作值来运行，它是能够表示并操作的值的类型

  1. 三种基本数据类型
    • 数字 量化工具
    • 字符串 用 '_' 或 "_" 包含的文本
    • 布尔值
  2. 两种小数据类型
    • null
    • undefined
  3. object类型
    • json对象 已命名的值的无序集合
    • 数组 有编号的值的有序集合
    • 函数 具有可执行代码，可通过调用来执行操作

类型转换

  1. 数字转字符串
    • 自动转换 var 字符串 = 数字 + "" ;
    • 指定小数位数 var 字符串 = 数字.toFixed(小数位数) ;
    • 指定有效数字位数 var 字符串 = 数字.toPrecision(有效数字位数) ;
    • 指定指数表示中的小数位数 var 字符串 = 数字.toExponential(指数表示中的小数位数) ;
