正则表达式：处理文本匹配的对象

  • 创建
    • 静态 var regExp = /正则表达式/[修饰符]
    • 动态 var regExp = new RegExp(正则表达式,[修饰符])
  • 修饰符
    • g 全局匹配（找所有的）
    • i 忽略大小写
    • m 多行匹配
  • RegExp属性
    • regExp.global 是否全局匹配（返回true / false）
    • regExp.ignoreCase 是否忽略大小写（返回true / false）
    • regExp.multiline 是否多行匹配（返回true / false）
    • regExp.source 正则表达式的原始文本
    • regExp.lastIndex 下一次匹配的起始位置
  • RegExp方法
    • regExp.compile 修改正则表达式后，重新编译
    • regExp.exec 匹配文本（返回找到的数组，第一个元素的位置）
    • regExp.test 匹配文本（返回true / false）
