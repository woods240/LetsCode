语句：指挥计算机干活的最基本指令

  • 顺序
    • 表达式 数学运算和函数调用
    • 语句块 { ... }
  • 分支
    • if ... else if ... else ... 含优先级、通过布尔值来判断
    • switch() { case: ... } 不含优先级、通过值匹配来判断
  • 循环
    • while() 条件循环
    • for( ; ; ) 计数器循环
    • for(key in object) 集合枚举（key是集合元素的键）
  • 跳转
    • break 跳出循环或switch
    • continue 开始下一次循环
    • return 跳出函数
  • 异常处理
    • throw() 抛出异常
    • try ... catch ... finally ... 捕获异常

注：
1.结构化程序的3种基本结构：顺序、分支、循环
2.数组和对象都可以看成键值对集合，数组将下标编号编号作为键、对象将属性名称作为键
3.异常是一个信号，通知运行环境，程序发生了意外
