给jQuery这个类，扩展静态方法

  • 直接定义 $.插件名称 = function () { //代码 }
  • 用方法扩展 $.extend { 插件名称 : function () { //代码 } }
  • 同时定义多个 $.命名空间 = { 插件1 : function () { //代码 }, 插件2 : function () { //代码 } }

注：
1.插件名称就是静态方法的名字，可以包含命名空间：命名空间.插件名称
2.函数也是对象，所以第一种方式可以转化为第三种方式
3.waitingPrompt.js是一个典型的类插件，请参考其代码
