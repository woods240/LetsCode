给jQuery这个类，扩展实例方法

  • 直接定义 $.fn.插件名称 = function () { //代码 }
  • 用方法扩展 $.fn.extend { 插件名称 : function () { //代码 } }
  • 同时定义多个 $.fn.命名空间 = { 插件1 : function () { //代码 }, 插件2 : function () { //代码 } }

注：
1.对象插件与类插件的定义相比，只是多了个 .fn
2.插件的目的是逻辑复用，两种插件仅是挂接的位置不同
3.逻辑面向jQuery元素的，使用对象插件； 逻辑更通用的，使用类插件
4.location.js是一个典型的对象插件，请参考其代码
